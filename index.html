About

  • CSV stands for comma-separated values. CSV file stores tabular data (numbers and text) in plain text. Upload or copy/paste CSV text for start.
  • Each line of CSV file is a data record consists of one or more fields. Fields separated by commas or semicolon for russian version. You can set custom separator.
  • You can output array, hash or hash with id. If you select hash, the hash key will be the first column.
  • You can view as string, readability string or JSONP.
